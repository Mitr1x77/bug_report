| **Описание** | После добавления товара в корзину, нет возможности удалить товар из корзины на странице с товарами |
| **Проект** | |
| **URL** | https://www.saucedemo.com/ |

| **Шаги** | 1. Перейти на сайт по ссылке - https://www.saucedemo.com/<br>2. Ввести данные пользователя, в поле Username: error_user, в поле Password: secret_sauce<br>3. Нажать на зеленую кнопку под названием Login<br>4. Выбрать 1 товар и нажать под ним кнопку - add to cart<br>5. Нажать на появившуюся под товаром кнопку remove |
| **Фактический результат** | Невозможность убрать товар из корзины нажатием на кнопку remove под товаром после добавления его в корзину |
| **Ожидаемый результат** | Успешное нажатие кнопки remove и удаление товара из корзины |
